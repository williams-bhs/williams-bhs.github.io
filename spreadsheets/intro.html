Welcome to a Spreadsheets Tutorial!

Notice that every row is numbered and every column has a letter. This provides a way to reference each cell. Click on C3.

Use the keyboard arrows to move around the spreadsheet. Hold Shift while using the arrows. Experiment with the Tab and Enter keys. Try entering values into some cells.

As you work, use the keyboard as much as you can.

Try the following:

  • Put a 3 into D4
  • Put cat into B7
  • Put 5/7 into A2

Once you're done, check your work.
