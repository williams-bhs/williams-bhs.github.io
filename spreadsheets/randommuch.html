More Randomness

What if we want a random number in a range, say from 40 to 50?

Using rand() generates a random decimal between 0 and 1, but we can multiply that and offset it!

In column A, fill the first 20 rows with random numbers between 40 and 50. Use:

=40+10*rand()

In column B, fill the first 20 rows with random numbers between 100 and 150. (hint: this is 10 + some amount of 50 )
