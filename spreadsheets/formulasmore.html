Formulas - Dolla sign y'all!

When we write the name of a cell like A2 into a formula, it is a reference to the value in that cell.

When we drag horizontally left-right, the column letter gets automatically updated.

When we drag vertically up-down, the row number gets automatically updated.

But what if we want the reference to stay the same?

=(4/5)*10+13

When writing fractions that have multiple terms, group terms with parentheses. The expression \[\frac{5-1}{2+3}\] is written in a formula as:

=(5-1)/(2+3)

Exponents use the symbol ^, i.e., \(2^{5}\) is =2^5

  1. In A1, write the formula for \(1+2+3\)
  2. In B1, write the formula for \(\frac{2^3}{4+1}\)
  3. In C1, write the formula for \(\left(\frac{1+7}{4^2}\right)^3\)
